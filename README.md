# Bloodcraft Images

Public repository for Bloodcraft mod pet/boss images.

## Structure

- `Images/pets/` - Boss/Pet images used by Discord webhook notifications

## Usage

Raw image URLs:
```
https://raw.githubusercontent.com/duugagno/Bloodcraft-Images/main/Images/pets/{filename}
```

Example:
```
https://raw.githubusercontent.com/duugagno/Bloodcraft-Images/main/Images/pets/1.loboalfa.jpg
```
